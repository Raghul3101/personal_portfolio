Self Balancing Robot

The Self-Balancing robot utilizes an Arduino Uno microcontroller, an MPU6050 accelerometer and gyroscope module, and various complementary components. This innovative project aims to achieve dynamic stability by constantly monitoring the robot's tilt angles using data from the MPU6050 sensor. The Arduino Uno processes this information and adjusts the motor speeds accordingly, ensuring the robot maintains an upright position..

Gesture Controlled Robot

The gesture-controlled robot, crafted with Arduino Uno, MPU6050, and a set of receiver and transmitter modules, seamlessly combines sensor technology with robotics. Harnessing the power of motion sensing, this innovation allows users to intuitively control the robot's movements through simple hand gestures, offering an immersive and interactive robotic experience..

Parking Management System

The Python-based Parking Management System, built with Tkinter and other graphical elements, revolutionizes vehicle management. The intuitive interface enhances user experience, providing a seamless and automated solution for efficient and secure parking management.
